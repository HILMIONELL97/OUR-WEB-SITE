# Guide de style pour le front-end

## Mise en page

Les dessins ont été créés dans les largeurs suivantes :

- Mobile : 375px
- Bureau : 1440px
- tablete : 800px

## Couleurs

### Primaire

- move : hsl(263, 49%, 52%)

### Neutre

- Bleu grisâtre : hsl(229, 6%, 66%)
